# WaterInBox.esp32
Aqui será armazenado arquivos referentes a um projeto realizado na matéria de Performance em Sistemas Ciberfísicos da Pontifícia Universidade Católica do Paraná.

## Objetivo
A meta é criar um dispositivo que será instalado dentro das caixas d'água, o qual medirá constantemente a entrada e saída de água. Registrando via ciber, todas as medidas registradas com a disponibilidade de um gráfico.

## Na prática + Ideias e futuras implementações
O dispositivo enviará semanalmente um gráfico ao smartphone do usuário, detalhando o consumo e a economia de água do reservatório.
